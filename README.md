# auto-loot
tera-proxy module to loot items automatically

## Dependency
- `Command` module

## Usage
### `loot` · `ㅣㅐㅐㅅ`
- Toggle on/off
- Default is on
### `loot auto` · `loot a`
- Toggle automatic loot attempt on interval on/off
- Default is on
### `loot status` · `loot s`
- Send status of module and auto function
